[console with Log enabled]


This image show the enabled log with colors.
There are two options to set
>log-file: true means All the logs will be saved in log file.
>log-settings: {
"info": true,
"debug": true,
"error": true,
"setup": true
}
Here you can choose which messages to show in development and production environment.
Keys with false value will be ignored and will not be showed in console.
